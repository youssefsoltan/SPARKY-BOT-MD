## `For Termux/Ssh/Ubuntu`
```bash
apt update
apt upgrade
pkg update && pkg upgrade
pkg install bash
pkg install libwebp
pkg install git
pkg install nodejs
pkg install ffmpeg
pkg install wget
pkg install imagemagick
git clone https://github.com/Sparkymon777/SPARKY-BOT-MD
cd SPARKY-BOT-MD
npm start
```
